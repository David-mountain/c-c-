# 链栈

## 原始模板

```c
#include <stdio.h>
#include <stdlib.h>

typedef int dataType;
typedef struct node 
{
	dataType data;
  	struct node * pNext;
} LinkStack;
```

## 入栈

```c
LinkStack* PushLkStack (LinkStack *top, dataType, x)
{
  LinkStack *p;
  p = (LinkStack*)malloc(sizeof(LinkStack));
  p->data = x;
  p->pNext = top;
  top = p;
  return top;
}
```



## 出栈

```c
LinkStack* PopLkStack (LinkStack *top, dataType *pData)
{
	LinkStack *p;
  	if ( top != NULL )
    {
      *pData = top->data;
      p = top;
      top = p->pNext;
      free(p);
    }
}
```
